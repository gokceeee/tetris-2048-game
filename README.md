# Tetris 2048 Game

A Python-based Tetris 2048 game developed as a team project for the COMP204 Programming Studio course.

## Features
- 2048 merge mechanics
- Chain merging system
- Save/load functionality
- Difficulty levels
- Pause system
- Ghost piece visualization
- Merge animations

## Team Project
This project was developed collaboratively as part of a 3-person team.
