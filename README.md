# Tetris 2048 Game

A Python-based Tetris 2048 game developed for the COMP204 Programming Studio course.

## Features
- 2048 merge mechanics
- Chain merging system
- Save/load functionality
- Difficulty levels
- Pause system
- Ghost piece visualization
- Merge animations
